# create-flexireact

Create FlexiReact v4 apps with one command.

## Usage

```bash
npx create-flexireact@latest my-app
```

Or with npm:

```bash
npm create flexireact@latest my-app
```

## Templates

| Template | Description |
|----------|-------------|
| **Default** | Full-featured with `routes/`, components, and Tailwind v4 |
| **Minimal** | Bare minimum FlexiReact setup |
| **App Router** | Next.js style `app/` directory routing |
