File-Based Routing

FlexiReact uses a powerful file-based routing system inspired by Next.js App Router

App Router (Recommended)

The app/ directory uses a nested folder structure where each folder represents a route segment.

Folder Structure 
app/
├── page.tsx           → /
├── layout.tsx         → Root layout
├── about/
│   └── page.tsx       → /about
├── blog/
│   ├── page.tsx       → /blog
│   └── [slug]/
│       └── page.tsx   → /blog/:slug
└── dashboard/
    ├── layout.tsx     → Dashboard layout
    ├── page.tsx       → /dashboard
    └── settings/
        └── page.tsx   → /dashboard/settings

page.tsx

Defines the UI for a route segment

layout.tsx

Shared UI that wraps child pages

loading.tsx

Loading UI with Suspense

error.tsx

Error boundary UI

Routes Directory (Alternative)

The routes/ directory provides a simpler file-based routing approach (Standard in v4).

Folder Structure 
routes/
├── index.tsx          → /
├── about.tsx          → /about
├── blog/
│   ├── index.tsx      → /blog
│   └── [slug].tsx     → /blog/:slug
└── api/
    └── users.ts       → /api/users

Dynamic Routes

Use square brackets to create dynamic route segments.

app/blog/[slug]/page.tsx 
interface PageProps {
  params: { slug: string };
}

export default function BlogPost({ params }: PageProps) {
  return (
    <div>
      <h1>Post: {params.slug}</h1>
    </div>
  );
}
ℹ️

Catch-All Routes

Use [...slug] for catch-all routes or [[...slug]] for optional catch-all.

Route Groups

Use parentheses to organize routes without affecting the URL structure.

Example 
app/
├── (marketing)/
│   ├── about/page.tsx     → /about
│   └── contact/page.tsx   → /contact
└── (shop)/
    ├── products/page.tsx  → /products
    └── cart/page.tsx      → /cart

API Routes

Create API endpoints by exporting HTTP method handlers.

app/api/users/route.ts 
export async function GET(request: Request) {
  const users = await db.users.findMany();
  return Response.json(users);
}

export async function POST(request: Request) {
  const body = await request.json();
  const user = await db.users.create({ data: body });
  return Response.json(user, { status: 201 });
}

Supported Methods

GET - Fetch data
POST - Create data
PUT - Update data
DELETE - Delete data
PATCH - Partial update

Request Helpers

request.json()
request.formData()
request.headers
request.cookies

Navigation

Link Component

Client-side navigation 
import { Link } from '@flexireact/core';

<Link href="/about">About</Link>
<Link href="/blog/my-post" prefetch>Blog Post</Link>

useRouter Hook

Programmatic navigation 
import { useRouter } from '@flexireact/core';

function MyComponent() {
  const router = useRouter();
  
  const handleClick = () => {
    router.push('/dashboard');
    // router.replace('/login');
    // router.back();
  };
  
  return <button onClick={handleClick}>Go</button>;
}
