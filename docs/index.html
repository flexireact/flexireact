Introduction

FlexiReact is a modern React framework designed for building fast, scalable web applications. It combines the best features of Next.js, Remix, and Astro into a simpler, more powerful package.

Installation

Create a new FlexiReact project with a single command:

Terminal

cd my-app
npm run dev

FlexiUI ‚Äî 50+ Components

A complete UI library with beautiful, accessible React components.

Installation 
npm install @flexireact/flexi-ui
Usage 
import { Button, Card, Input, Modal, Toast } from '@flexireact/flexi-ui';

function MyComponent() {
  return (
    <Card className="p-6">
      <Input placeholder="Enter your email" />
      <Button>Subscribe</Button>
    </Card>
  );
}
