Layouts & Templates

Create shared UI that persists across routes with layouts and templates

Root Layout

The root layout is required and wraps all pages in your application.

app/layout.tsx 
import React from 'react';

export default function RootLayout({ 
  children 
}: { 
  children: React.ReactNode 
}) {
  return (
    <html lang="en">
      <head>
        <meta charSet="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <link rel="stylesheet" href="/styles.css" />
      </head>
      <body>
        <header>My App</header>
        <main>{children}</main>
        <footer>© 2024</footer>
      </body>
    </html>
  );
}
⚠️

Important

The root layout must include <html> and <body> tags.

Nested Layouts

Create layouts for specific route segments that inherit from parent layouts.

app/dashboard/layout.tsx 
import React from 'react';
import { Sidebar } from './components/Sidebar';

export default function DashboardLayout({ 
  children 
}: { 
  children: React.ReactNode 
}) {
  return (
    <div className="flex">
      <Sidebar />
      <div className="flex-1">
        {children}
      </div>
    </div>
  );
}
Folder Structure 
app/
├── layout.tsx              → Root layout (all pages)
├── page.tsx                → /
└── dashboard/
    ├── layout.tsx          → Dashboard layout
    ├── page.tsx            → /dashboard
    └── settings/
        └── page.tsx        → /dashboard/settings
ℹ️

Layout Nesting

Layouts nest automatically. The /dashboard/settings page will be wrapped by both the root layout and dashboard layout.

Templates

Templates are similar to layouts but create a new instance on each navigation.

app/template.tsx 
import React from 'react';

export default function Template({ 
  children 
}: { 
  children: React.ReactNode 
}) {
  return (
    <div className="animate-fade-in">
      {children}
    </div>
  );
}

Layouts

  • ✓ State persists across navigation
  • ✓ Don't remount on navigation
  • ✓ Better for shared UI

Templates

  • ✓ New instance on each navigation
  • ✓ Remount on navigation
  • ✓ Better for animations/effects

Metadata

Define metadata for SEO in layouts and pages.

Static Metadata 
export const metadata = {
  title: 'My App',
  description: 'Welcome to my app',
  keywords: ['react', 'flexireact', 'ssr'],
  openGraph: {
    title: 'My App',
    description: 'Welcome to my app',
    images: ['/og-image.png'],
  },
};
Dynamic Metadata 
export async function generateMetadata({ params }) {
  const post = await getPost(params.slug);
  
  return {
    title: post.title,
    description: post.excerpt,
    openGraph: {
      images: [post.coverImage],
    },
  };
}

Loading States

Create instant loading states with loading.tsx.

app/dashboard/loading.tsx 
export default function Loading() {
  return (
    <div className="flex items-center justify-center h-screen">
      <div className="animate-spin rounded-full h-12 w-12 border-t-2 border-primary"></div>
    </div>
  );
}
ℹ️

Automatic Suspense

FlexiReact automatically wraps your page in a Suspense boundary with the loading UI as fallback.

Error Handling

Handle errors gracefully with error.tsx.

app/dashboard/error.tsx 
'use client';

import React from 'react';

export default function Error({ 
  error, 
  reset 
}: { 
  error: Error; 
  reset: () => void 
}) {
  return (
    <div className="p-8">
      <h2 className="text-2xl font-bold mb-4">Something went wrong!</h2>
      <p className="text-gray-400 mb-4">{error.message}</p>
      <button onClick={reset} className="px-4 py-2 bg-primary text-black rounded">
        Try again
      </button>
    </div>
  );
}

Not Found Pages

Create custom 404 pages with not-found.tsx.

app/not-found.tsx 
import React from 'react';
import Link from '@flexireact/core';

export default function NotFound() {
  return (
    <div className="flex flex-col items-center justify-center h-screen">
      <h1 className="text-6xl font-bold mb-4">404</h1>
      <p className="text-gray-400 mb-8">Page not found</p>
      <Link href="/" className="px-6 py-3 bg-primary text-black rounded">
        Go Home
      </Link>
    </div>
  );
}
